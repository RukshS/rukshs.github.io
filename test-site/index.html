Mozilla Firefox

Mozilla Firefox is a free and open-source web browser developed by Mozilla Foundation and its subsidiary, Mozilla Corporation. Firefox was created as a faster, more private alternative to browsers like Internet Explorer and Chrome. You can download Firefox for desktop from Mozilla's website.
